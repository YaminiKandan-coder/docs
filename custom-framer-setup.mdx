## Overview

Use this setup if you offer **made-to-order framing** or print services where customers need to **upload their own image**.

<Callout type="info" title="Best for">
Custom framers and print-on-demand sellers supporting customer image uploads.
</Callout>

---

## Configuration Workflow

<Steps>
  <Step title="1. Open Components">
    In your Shopify admin, open the **LIVEARf app**, then go to **Components**.
  </Step>

  <Step title="2. Select Upload Component">
    Choose **Upload Component** (or create a new one if needed).
  </Step>

  <Step title="3. Confirm Image Upload is Enabled">
    Ensure the **Image Upload** toggle is ON so customers can upload images on the product page.
  </Step>

  <Step title="4. Configure Frame + Mat Options">
    Set up:
    - Frames
    - Mat Size
    - Mat Colour  
    (and any additional custom options you support)
  </Step>

  <Step title="5. Validate Customer Flow">
    Confirm both flows work:
    - Upload Image (primary)
    - Default Preview before upload (optional)
  </Step>

  <Step title="6. Save & Link">
    Click **Save**, then link the component to products using the LIVEVUE template.
  </Step>
</Steps>
